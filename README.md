PROCESSO DE CONTRIBUIÇÃO PARA O REPOSITÓRIO


## Fluxo do Aluno

### 1. Preparação (Apenas primeira vez)

* Baixar e instalar o Git
* Verificar instalação:
```bash
git --version

```



### 2. Clonar e Configurar

* Clonar o repositório:

```bash
  git clone <URL_DO_REPOSITORIO>

```

* Entrar na pasta criada:
```bash
cd nome_da_pasta

```


* Vincular seu usuário ao repositório (evita erro de permissão):

```bash
  git remote set-url origin "https://seu_usuario@github.com/angelabenet/TESTE_G.git"

```

* Abrir no VS Code:

```bash
  code .

```

### 3. Alteração e Envio

* Fazer a alteração no código (ex: `app.py`)
* Ver o que mudou:
```bash
git status

```


* Adicionar o arquivo alterado:
```bash
git add app.py

```


* Salvar a alteração com uma mensagem:

```bash
  git commit -m "alteração 1"

```

* Enviar para o GitHub:
```bash
git push

```


* Entrar no seu GitHub e checar se a alteração subiu.

### Revisão e Aceite

* Ir na aba **Pull Requests** $\rightarrow$ **New pull request**
* Conferir o nome e o que foi alterado no código
* Clicar em **Merge pull request** para aceitar a alteração
